ShopperStop Promotional Pricing API

Interactive API reference · Version 1.0.0 · Calculator · OpenAPI JSON

Getting started

Expand an operation, update the sample request if needed, and select Execute. All calls run against this local service. Promotion values use the field value; create a promotion and activate it before it is applied to bills.

System

GET/health

Reports service availability.


    
Bill controller

    
POST/api/v1/bills/calculate
Calculates progressive customer-tier and active-promotion discounts for a cart.

    
Promotion controller

    
GET/api/v1/promotions
Lists configured promotions. Add ?active=true to list active records.

    
POST/api/v1/promotions
Creates an inactive promotion. Types: FLAT, PERCENTAGE, CATEGORY, BUY_X_GET_Y, TIME.

    
PUT/api/v1/promotions/{id}
Updates an existing promotion. Replace {id} with an ID from the list response.

    
DELETE/api/v1/promotions/{id}
Soft-deletes a promotion.

    
POST/api/v1/promotions/{id}/activate
Activates a configured promotion.

    
POST/api/v1/promotions/{id}/deactivate
Deactivates a configured promotion.

    
POST/api/v1/promotions/simulate
Previews an unsaved promotion against a bill without storing it.

    
Customer tier controller

    
GET/api/v1/customer-tiers
Returns configured customer tier slabs.

    
POST/api/v1/customer-tiers
Creates or replaces a custom tier definition.
